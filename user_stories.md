# User Stories – Smart Clinic Management System

---

## Admin User Stories

### US-ADM-01: Manage Doctors
**As an Admin**, I want to add and manage doctors so that patients can book appointments.

**Acceptance Criteria**
- Admin can add a doctor with name, email, and specialization
- Admin can view the list of all doctors
- Admin can update doctor details

**Priority:** High  
**Story Points:** 5

---

### US-ADM-02: View Appointments
**As an Admin**, I want to view all appointments so that clinic operations are monitored.

**Acceptance Criteria**
- Admin can view appointments by date
- Admin can see doctor and patient details for each appointment

**Priority:** Medium  
**Story Points:** 3

---

### US-ADM-03: Manage Patient Records
**As an Admin**, I want to manage patient records to ensure accurate data.

**Acceptance Criteria**
- Admin can view patient details
- Admin can update patient information

**Priority:** Medium  
**Story Points:** 3

---

## Doctor User Stories

### US-DOC-01: View Appointments
**As a Doctor**, I want to view my scheduled appointments so that I can plan my day.

**Acceptance Criteria**
- Doctor can see only their own appointments
- Appointments show patient name and time

**Priority:** High  
**Story Points:** 5

---

### US-DOC-02: Add Prescription
**As a Doctor**, I want to add prescriptions for patients after consultation.

**Acceptance Criteria**
- Doctor can add medicine name and dosage
- Prescription is linked to an appointment

**Priority:** High  
**Story Points:** 5

---

### US-DOC-03: View Patient History
**As a Doctor**, I want to view patient history to provide better treatment.

**Acceptance Criteria**
- Doctor can view previous appointments
- Doctor can view past prescriptions

**Priority:** Medium  
**Story Points:** 3

---

## Patient User Stories

### US-PAT-01: Register and Login
**As a Patient**, I want to register and log in so that I can access clinic services.

**Acceptance Criteria**
- Patient can register using email and password
- Patient can log in successfully with valid credentials

**Priority:** High  
**Story Points:** 5

---

### US-PAT-02: Book Appointment
**As a Patient**, I want to view available doctors and book appointments.

**Acceptance Criteria**
- Patient can view doctors by specialization
- Patient can book an available time slot

**Priority:** High  
**Story Points:** 5

---

### US-PAT-03: View Prescriptions
**As a Patient**, I want to view my prescriptions and appointment history.

**Acceptance Criteria**
- Patient can view prescriptions linked to appointments
- Patient can see appointment history

**Priority:** Medium  
**Story Points:** 3
